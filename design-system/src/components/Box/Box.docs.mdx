---
title: Box
description: Low level primitive for creating components
---

```tsx
import { Box } from "design-system";
```

```tsx expand=true live=true
<Box padding={3} backgroundColor="link">
  Hello world
</Box>
```

## Variants

```tsx live=true
<Box display="flex" flexDirection={{ small: "column", medium: "row" }}>
  <Button variant="primary">Responsive</Button>
  <Button variant="secondary">Responsive</Button>
</Box>
```
